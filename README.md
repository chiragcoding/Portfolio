## ✨ What Is This?

A fully immersive **3D portfolio website** built from scratch — no templates, no page builders. Navigate through interactive 3D scenes using scroll, touch, or keyboard.

## 🎮 Sections

| # | Section | What's There |
|---|---------|-------------|
| 1 | **Home** | Holographic data core with orbital rings, particles, and floating shapes |
| 2 | **Experience** | Animated data pipeline with interactive nodes — click to explore achievements |
| 3 | **Projects & Skills** | 3D project cards + rotating skills sphere |
| 4 | **Education** | Academics, certifications, and achievements |

## 🛠️ Tech Stack

```
Frontend        → React 18 + TypeScript
3D Engine       → Three.js + React Three Fiber + Drei
State           → Zustand
Build           → Vite
Deployment      → Vercel
```

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd portfolio-3d
npm install
npm run dev
```

Open `http://localhost:5173`

## 📁 Structure

```
src/
├── scenes/          → 3D scenes (Landing, Pipeline, ServerRoom, Education)
├── components/      → Camera, Navigation, Contact, Loading, Fallback
├── store/           → Zustand state management
├── hooks/           → Navigation & responsive hooks
├── data/            → All content (projects, skills, education)
└── types/           → TypeScript interfaces
```
